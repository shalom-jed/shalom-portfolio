# Shalom Jedidiah - Personal Portfolio

A sleek, responsive personal portfolio website designed to showcase my software engineering projects, experience, and academic journey. Built with a modern "Black and Tech Blue" aesthetic, featuring interactive UI elements and dynamic content rendering.

**[View Live Site](#)** *(https://shalom-jed.github.io/shalom-portfolio/)*

## ‚ú® Features

* **Interactive Custom Cursor:** A custom tracking dot and blob effect that follows mouse movement for a premium feel.
* **Dynamic Project Rendering:** Projects are automatically mapped and generated from a centralized data file (`app/data/projects.ts`), complete with auto-selected icons based on project keywords.
* **Split Timeline UI:** Distinct, side-by-side tracking of professional Experience and Academic history.
* **Fully Responsive:** Includes a custom mobile hamburger navigation menu and optimized UI scaling for all device sizes.
* **Modern Tech Aesthetic:** A deep black and tech-blue color palette with glassmorphism effects and mesh gradients.
