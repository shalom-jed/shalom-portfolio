# Shalom Jedidiah - Portfolio

Personal portfolio showcasing projects, experience, and technical skills.

## 🌐 Live Site

**https://shalom-jed.github.io/shalom-portfolio/**

## ✨ Features

- Interactive UI with custom cursor
- Dynamic project rendering
- Experience & education timeline
- Responsive design
- Dark mode aesthetic

## 💻 Tech Stack

Next.js • React • TypeScript • Tailwind CSS • Lucide React

## 🚀 Local Development
```bash
git clone https://github.com/shalom-jed/shalom-portfolio.git
cd shalom-portfolio
npm install
npm run dev
```

## 📦 Build & Deploy
```bash
npm run build
npm run export
```

Deployed on GitHub Pages.
